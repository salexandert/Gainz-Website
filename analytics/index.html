Website transparency

Privacy-preserving analytics for the public website.

Gainz uses aggregate public-site feedback, such as page visits and release downloads, without tracking local app users, transaction history, tax evidence, or audit packets.

Scope

Analytics described here apply to cryptogainz.store and public GitHub release counts, not the local Gainz desktop app.

Current approach

Aggregate website analytics only.

Website visits

Netlify Web Analytics can report aggregate page views and referrers from hosting logs without adding a client-side tracking script to this repo.

Release downloads

GitHub Release asset download counts are public aggregate numbers. Gainz snapshots those counts daily in the app repository.

No app telemetry

The local Gainz app does not upload transaction history, tax evidence, audit packets, or usage events to the website.

Offline boundary

Website analytics apply to cryptogainz.store, not to reconciliation work performed on your computer.
