Workflow

How to approach missing basis.

  1. Import every exchange CSV and wallet/export file you can locate.
  2. Declare current holdings so Gainz can compare imported activity with what you actually hold.
  3. Review the automatic FIFO result. Gainz uses documented acquisition lots before any unsupported-basis treatment.
  4. Use the Guided Review Queue to work missing-basis items one at a time.
  5. Select a treatment, review its exact proceeds, basis, holding-period, and Form 8949 impact, then explicitly apply it only when supported by recorded professional direction.
  6. Generate a packet that shows imported facts, fees, applied assumptions, receipts, reversals, and anything still needing research.
When records cannot be recovered

Document the boundary between facts and assumptions.

Use documented basis first

Gainz preserves linked FIFO lots and isolates only the exact unsupported sale quantity.

Review the impact

The preview shows source gross value, allocated fee, net proceeds, basis, gain, and Form 8949 totals before anything changes.

Record professional direction

The user records the professional name, role, date, evidence, and treatment. Gainz does not verify identity or credentials.

Keep it reversible

Applied treatments create an identified adjustment, calculation receipt, packet workpaper, and reversal path.

Not tax advice
